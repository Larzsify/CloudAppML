Websockets
Open an interactive communication seession between the user and server.
With the API, messages can be sent to the server and you can receive event-driven responses without having to poll the server for reply

Events:
CloseEvent
Event sent by websocket object when connection closes
MessageEvent
Event sent by websocket object when a message is received from server777

APIs
Socket.io --> Node.js (cross-platform)
WebSocket-Node --> Node.js (WebSocket server API implementation) 
Total.js --> Node.js (Web application framework)
Faye --> Node.js (Websocket 2-way) (EventSource one-way)
